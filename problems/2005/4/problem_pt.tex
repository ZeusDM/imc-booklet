Determine todos os polinômios \(P(x) = a_n x^n + a_{n-1} x^{n-1} + \ldots + a_1 x + a_0\) (\(a_n \ne 0\)) que satisfazem as seguintes duas condições:
\begin{itemize}
    \item[(i)] \((a_0,a_1,\ldots,a_n)\) é uma permutação dos números \((0,1,\ldots,n)\); e
    \item[(ii)] todas as raízes de \(P(x)\) são números racionais.
\end{itemize}
