Para um inteiro \(n \ge 3\), considere os conjuntos
\begin{align*}
S_n &= \{ (x_1,x_2,\ldots,x_n) : x_i \in \{0, 1, 2\} \text{ para todo } i \le n \}; \\
A_n &= \{(x_1,x_2,\ldots,x_n) \in S_n : |\{x_i, x_{i+1}, x_{i+2}\}| \ne 1 \text{ para todo } i \le n - 2\}; \text{ e} \\
B_n &= \{(x_1,x_2,\ldots,x_n) \in S_n : (x_i = x_{i+1} \Rightarrow x_i \ne 0) \text{ para todo } i \le n - 1\} .
\end{align*}
Prove que \(|A_{n+1}| = 3 \cdot |B_n|\).
