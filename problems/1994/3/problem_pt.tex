Dado um conjunto \(S\) de \(2n-1\), \(n\in \mathbb{N}\), números irracionais diferentes. Prove que existem \(n\) elementos diferentes \(x_1, x_2, \ldots, x_n \in S\) tais que para todos os números racionais não negativos \(a_1, a_2, \ldots, a_n\) com \(a_1 + a_2 + \ldots + a_n > 0\) temos que \(a_1 x_1 + a_2 x_2 + \cdots + a_n x_n\) é um número irracional.
