Seja \(f\) uma função de valor real com \(n+1\) derivadas em cada ponto de \(\mathbb R\). Mostre que para cada par de números reais \(a\), \(b\), \(a<b\), tal que
\[
    \ln\left( \frac{f(b)+f'(b) + \cdots + f^{(n)} (b)}{f(a) + f'(a) + \cdots + f^{(n)}(a)}\right) = b-a
\]
existe um número \(c\) no intervalo aberto \((a,b)\) para o qual
\[
    f^{(n+1)}(c) = f(c).
\]
