\begin{itemize}
    \item[(a)] Seja \(A\) uma matriz \(n\times n\), \(n\geq 2\), simétrica, invertível, com elementos reais positivos. Mostre que \(z_n\leq n^2-2n\), onde \(z_n\) é o número de elementos zero em \(A^{-1}\).

    \item[(b)] Quantos elementos zero existem na inversa da matriz \(n\times n\)
    \[ A = \begin{pmatrix}
        1&1&1&1&\ldots&1\\
        1&2&2&2&\ldots&2\\
        1&2&1&1&\ldots&1\\
        1&2&1&2&\ldots&2\\
        \vdots&\vdots&\vdots&\vdots&\ddots&\vdots\\
        1&2&1&2&\ldots&\ddots
    \end{pmatrix}. \]
\end{itemize}
