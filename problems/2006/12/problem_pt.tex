Sejam \(A_{i},B_{i},S_{i}\) (\(i\in\{1,2,3\}\)) matrizes reais \(2\times 2\) invertíveis tais que
\begin{itemize}
	\item nem todas as \(A_{i}\) têm um autovetor real comum,
	\item \(A_{i}=S_{i}^{-1}B_{i}S_{i}\) para \(i\in\{1,2,3\}\),
	\item \(A_{1}A_{2}A_{3}=B_{1}B_{2}B_{3}=I\).
\end{itemize}
Prove que existe uma matriz \(2\times 2\) invertível \(S\) tal que \(A_{i}=S^{-1}B_{i}S\) para todo \(i\in\{1,2,3\}\).
