Dado um inteiro \(n>1\), seja \(S_n\) o grupo de permutações dos números \(1,\;2,\;3,\;\ldots,\;n\). Dois jogadores, A e B, jogam o seguinte jogo. Alternando turnos, eles selecionam elementos (um elemento por vez) do grupo \(S_n\). É proibido selecionar um elemento que já foi selecionado. O jogo termina quando os elementos selecionados geram todo o grupo \(S_n\). O jogador que fez a última jogada perde o jogo. A primeira jogada é feita por A. Qual jogador tem uma estratégia vencedora?
