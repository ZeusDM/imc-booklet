Let \(f\in C^3(\mathbb{R})\) nonnegative function with \(f(0)=f'(0)=0, f''(0)>0\). Define \(g(x)\) as follows:
\[ g(x)= \begin{cases} \left( \sqrt{f(x)}/f'(x) \right)' &\text{for } x\neq 0 \\ 0 &\text{for } x=0 \end{cases} \]
\begin{itemize}
\item[(a)] Show that \(g\) is bounded in some neighbourhood of \(0\).
\item[(b)] Is the above true for \(f\in C^2(\mathbb{R})\)?
\end{itemize}
