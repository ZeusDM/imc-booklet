Seja \(f\in C^3(\mathbb{R})\) uma função não negativa com \(f(0)=f'(0)=0\) e \(f''(0)>0\). Defina \(g(x)\) da seguinte forma:
\[ g(x)= \begin{cases} \left( \sqrt{f(x)}/f'(x) \right)' &\text{para } x\neq 0 \\ 0 &\text{para } x=0 \end{cases} \]
\begin{itemize}
\item[(a)] Mostre que \(g\) é limitada em algum vizinhança de \(0\).
\item[(b)] A afirmação acima é verdadeira para \(f\in C^2(\mathbb{R})\)?
\end{itemize}
