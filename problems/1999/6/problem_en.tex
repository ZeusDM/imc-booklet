\begin{itemize}
\item[(a)] Let \(p>1\) a real number. Find a real constant \(c_p\) for which the following statement holds:
If \(f\colon  [-1,1]\to\mathbb{R}\) is a continuously differentiable function with \(f(1)>f(-1)\) and \(|f'(y)|\le1 \forall y\in[-1,1]\), then \(\exists x\in[-1,1]: f'(x)>0\) so that \(\forall y\in[-1,1]: |f(y)-f(x)|\le c_p\sqrt[p]{f'(x)}|y-x|\).

\item[(b)] What if \(p=1\)?
\end{itemize}
