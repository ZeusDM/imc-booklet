Seja \(a\) um inteiro positivo par. Determine todos os números reais \(x\) tais que
\[
\left\lfloor \sqrt[a]{b^a + x \cdot b^{a-1}} \right\rfloor = b^a + \lfloor x/a \rfloor \tag{1}
\]
vale para todo inteiro positivo \(b\).

(Aqui \(\lfloor x \rfloor\) denota o maior inteiro que não é maior que \(x\).)
