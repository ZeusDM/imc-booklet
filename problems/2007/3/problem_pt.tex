Chame um polinômio \( P(x_{1}, \ldots, x_{k})\) \textit{bom} se existirem matrizes reais \( 2\times 2\) \( A_{1}, \ldots, A_{k}\) tais que
\[P(x_{1}, \ldots, x_{k}) = \det \left(\sum_{i=1}^{k}x_{i}A_{i}\right).\]

Encontre todos os valores de \( k\) para os quais todos os polinômios homogêneos com \( k\) variáveis de grau 2 são bons. (Um polinômio é homogêneo se cada termo tem o mesmo grau total.)
