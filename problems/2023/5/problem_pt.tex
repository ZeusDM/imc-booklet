Considere inteiros positivos fixos \(n\) e \(k\) tais que \(2 \le k \le n\) e um conjunto \(M\) consistindo de \(n\) frutas. Uma \emph{permutação} é uma sequência \(x = (x_1, x_2, \ldots, x_n)\) tal que \(\{x_1, \ldots, x_n\} = M\). Ivan \emph{prefere} algumas (pelo menos uma) dessas permutações. Ele percebeu que para toda permutação preferida \(x\), existem \(k\) índices \(i_1 < i_2 < \ldots < i_k\) com a seguinte propriedade: para todo \(1 \le j < k\), se ele trocar \(x_{i_j}\) e \(x_{i_{j+1}}\), ele obtém outra permutação preferida.

Prove que ele prefere pelo menos \(k!\) permutações.
