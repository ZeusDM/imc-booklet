Sejam \(x_1,\ldots,x_n\) números reais. Para qualquer conjunto \(I\subset\{1,2,\ldots,n\}\) seja \(s(I)=\sum_{i\in I}x_i\). Suponha que a função \(I\to s(I)\) assume pelo menos \(1.8^n\) valores, onde \(I\) percorre todos os \(2^n\) subconjuntos de \(\{1,2,\ldots,n\}\). Prove que o número de conjuntos \(I\subset \{1,2,\ldots,n\}\) para os quais \(s(I)=2019\) não excede \(1.7^n\).
