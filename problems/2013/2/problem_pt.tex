Seja \(\displaystyle{f:{\cal R} \to {\cal R}}\) uma função duas vezes diferenciável. Suponha que \(\displaystyle{f\left( 0 \right) = 0}\). Prove que existe \(\displaystyle{\xi  \in \left( { - \frac{\pi }{2},\frac{\pi }{2}} \right)}\) tal que \[\displaystyle{f''\left( \xi  \right) = f\left( \xi  \right)\left( {1 + 2{{\tan }^2}\xi } \right)}.\]
