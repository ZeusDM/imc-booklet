Existem \(2n\) alunos em uma escola (\(n \in \mathbb{N}, n \geqslant 2\)). A cada semana, \(n\) alunos vão em uma viagem. Após várias viagens, a seguinte condição foi atendida: todos os pares de alunos estiveram juntos em pelo menos uma viagem. Qual é o número mínimo de viagens necessárias para que isso aconteça?
