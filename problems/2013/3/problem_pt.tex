Há \(2n\) estudantes em uma escola (\(n \in \mathbb{N},n \geqslant 2\)). A cada semana, \(n\) estudantes vão em uma viagem. Após várias viagens, a seguinte condição foi cumprida: cada dois estudantes estiveram juntos em pelo menos uma viagem. Qual é o número mínimo de viagens necessárias para que isso aconteça?
