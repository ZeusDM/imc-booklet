Prove ou disprova as seguintes afirmações:\begin{itemize}

\item[(a)] Existe uma função monótona \(f \colon  [0, 1] \to [0, 1]\) tal que para cada \(y \in [0, 1]\) a equação \(f(x) = y\) tem infinitas soluções \(x\).
\item[(b)] Existe uma função continuamente diferenciável \(f \colon  [0, 1] \to [0, 1]\) tal que para cada \(y \in [0, 1]\) a equação \(f(x) = y\) tem infinitas soluções \(x\).
\end{itemize}
