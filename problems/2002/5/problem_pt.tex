Prove ou disprova as seguintes afirmações:\begin{itemize}

\item[(a)] Existe uma função monótona \(f : [0, 1] \rightarrow [0, 1]\) tal que para cada \(y \in [0, 1]\) a equação \(f(x) = y\) tem incontáveis soluções \(x\).
\item[(b)] Existe uma função continuamente diferenciável \(f : [0, 1] \rightarrow [0, 1]\) tal que para cada \(y \in [0, 1]\) a equação \(f(x) = y\) tem incontáveis soluções \(x\).
\end{itemize}
