\begin{itemize}
\item[(a)] Uma sequência \(x_1,x_2,\dots\) de números reais satisfaz
\[x_{n+1}=x_n \cos x_n \textrm{ para todo } n\geq 1.\]
Isso implica que essa sequência converge para todos os valores iniciais \(x_1\)?

\item[(b)] Uma sequência \(y_1,y_2,\dots\) de números reais satisfaz
\[y_{n+1}=y_n \sin y_n \textrm{ para todo } n\geq 1.\]
Isso implica que essa sequência converge para todos os valores iniciais \(y_1\)?
\end{itemize}
