Denote por \(S_n\) o grupo de permutações da sequência \((1,2,\dots,n).\) Suponha que \(G\) é um subgrupo de \(S_n,\) tal que para cada \(\pi\in G\setminus\{e\}\) existe um único \(k\in \{1,2,\dots,n\}\) para o qual \(\pi(k)=k.\) (Aqui \(e\) é o elemento unitário do grupo \(S_n.\)) Mostre que esse \(k\) é o mesmo para todos \(\pi \in G\setminus \{e\}.\)
