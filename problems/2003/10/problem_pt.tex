Encontre todos os inteiros positivos \(n\) para os quais existe uma Família \(\mathcal{F}\) de subconjuntos de três elementos de \(S=\{1,2,\dots,n\}\) satisfazendo
\[\text{(i) para quaisquer dois elementos diferentes \(a,b \in S\) existe exatamente um \(A \in \mathcal{F}\) contendo ambos \(a\) e \(b\);}\]\[\text{(ii) se \(a,b,c,x,y,z\) são elementos de \(S\) tais que \(\{a,b,x\},\{a,c,y\},\{b,c,z\} \in \mathcal{F}\), então \(\{x,y,z\} \in \mathcal{F} \)  }.\]
