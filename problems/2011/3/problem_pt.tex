Seja \(p\) um número primo. Chamamos um inteiro positivo \(n\) de interessante se
\[x^n-1=(x^p-x+1)f(x)+pg(x)\]
para alguns polinômios \(f\) e \(g\) com coeficientes inteiros.\begin{itemize}

\item[(a)] Prove que o número \(p^p-1\) é interessante.
\item[(b)] Para quais \(p\) é \(p^p-1\) o menor número interessante?
\end{itemize}
