Sejam \(f,g\colon [a,b]\to [0,\infty)\) duas funções contínuas e não decrescentes tais que para cada \(x\in [a,b]\) temos
\[ \int^x_a \sqrt { f(t) }\ dt \leq \int^x_a \sqrt { g(t) }\ dt  \ \  \textrm{e}\ \int^b_a \sqrt {f(t)}\ dt = \int^b_a \sqrt { g(t)}\ dt. \]

Prove que
\[ \int^b_a \sqrt { 1+ f(t) }\ dt \geq \int^b_a \sqrt { 1 + g(t) }\ dt. \]
