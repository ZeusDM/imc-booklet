Dizemos que um número real positivo \(d\) é \emph{bom} se existe uma sequência infinita \(a_1,a_2,a_3,\dots\in (0,d)\) tal que para cada inteiro positivo \(n\), os pontos \(a_1,a_2,\dots,a_n\) partitionam o intervalo \([0,d]\) em segmentos de comprimento no máximo \(\frac{1}{n}\) cada. Encontre \(\text{sup}\{d : d  \text{ é  bom}\}\).
